About the Project

Messenger App is a simple real-time messaging application developed as part of the Software Engineering course at the University of Cincinnati. The project focuses on providing an easy and efficient way for users to communicate through private messaging. Throughout the semester, the application will evolve through iterative development and continuous feedback from users and stakeholders.

Current Features

  • Login with a unique username
  • Find and search for users
  • Send private messages
  • Receive private messages in real time

Additional features will be added throughout the semester as the project continues to evolve.
